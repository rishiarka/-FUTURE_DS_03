# Marketing Funnel & Conversion Analysis

This project analyzes user behavior across a marketing funnel to identify conversion rates and drop-off points.

## Objective
- Analyze user journey from product view to purchase
- Identify drop-offs at each stage
- Improve conversion strategies

## Tools Used
- Power BI
- Data Visualization
- Funnel Analysis

## Key Insights
- Significant drop-off between product views and cart additions
- Low conversion from cart to purchase stage
- Certain product categories perform better in conversions

## Recommendations
- Improve product page experience to increase engagement
- Offer discounts at cart stage
- Optimize checkout process
- Retarget users who abandon carts
